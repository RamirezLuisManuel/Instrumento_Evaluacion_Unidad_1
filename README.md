# IoT_instrumento_de_evaluacion_unidad_1

## Ejercicio practico "semaforo con inyeccion de datos del sensor en postgres"
### Descripcion
En este ejercicio se creo un circuito para poder leer los datos de un sensor de proximidad, y poder inyectar en una base de datos en SQLite los datos de la distancia que el sensor detecte, y con esta inyección de datos, 3 leds de distintos colores, enciendan o apaguen según un rango de distancia definido, tomando como base la distancia registrada por el sensor.

## Ejercico "Comunicacion con mosquito"
### Descripcion
